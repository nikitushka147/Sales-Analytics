# Sales-Analytics

Цей проєкт присвячений комплексному аналізу даних про продажі, об’єднанню розрізнених джерел інформації та створенню інтерактивних візуалізацій для бізнес-аналітики.

📋 Опис проєкту
Скрипт виконує повний цикл обробки даних (ETL):

Завантаження та очищення: Об’єднання даних із трьох джерел (країни, події/замовлення, товари). Обробка пропущених значень та корекція типів даних (дати, числові показники).

Розрахунок метрик: Автоматичне обчислення ключових бізнес-показників, таких як:

Виручка (Revenue), собівартість (Cost) та прибуток (Profit).

Маржинальність (Margin %).

Час доставки (Shipping days).

Глибокий аналіз: Групування даних за категоріями товарів, регіонами, каналами продажу та часовими інтервалами.

📊 Візуалізація даних
У проєкті використано бібліотеки Matplotlib, Seaborn та Plotly для створення:

Фінансових звітів: Порівняння прибутку та виручки за категоріями товарів.

Географічних карт: Інтерактивні карти (Choropleth maps), що показують розподіл прибутку по всьому світу.

Аналізу ефективності: Гістограми розподілу прибутку за каналами продажу та часу доставки.

Часових трендів: Лінійні графіки динаміки виручки для Топ-5 країн.

🛠 Стек технологій
Мова: Python

Бібліотеки: Pandas, NumPy

Візуалізація: Matplotlib, Seaborn, Plotly Express

Середовище: Google Colab / Jupyter Notebook
